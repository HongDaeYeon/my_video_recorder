# my_video_recorder

해당 프로젝트는 Opencv를 활용한 웹캠 화면 출력 및 동영상 녹화를 할 수 있는 Python 프로그램입니다.


# 기능
-   **녹화**:  `Space`  키를 눌러 녹화 시작/중지 (녹화 중에는 화면 좌상단에 빨간색 원 표시)
-   **명암 대비 및 밝기 조정**:  `C/V`  키로 대비 +/-,   `B/N`  키로 밝기 +/-
-   **좌우 반전**:  `F`  키를 눌러 화면 좌우 반전
-   **종료**:  `ESC`  키를 눌러 프로그램 종료
